## 🎮 游戏玩法

1. **WASD 移动**：  
   用你的键盘操控飞机，灵活躲避敌人！🕹️

2. **鼠标点击射击**：  
   用子弹告诉敌人谁才是宇宙的老大！🖱️

3. **空格键暂停**：  
   累了吗？按下空格键，喝口水再战！⏸️

4. **左 Shift 激光波**：  
   当敌人太多时，用激光波清场！⚡（但要省着点用哦~）
